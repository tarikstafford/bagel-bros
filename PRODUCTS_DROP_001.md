## Quick Add Instructions

### For Each Product:

1. **In Shopify Admin:**
   - Products â†’ Add product
   - Copy/paste title and description
   - Set price and compare-at price
   - Add cost per item (for profit tracking)

2. **Add Variants:**
   - Click "Add variant"
   - Enter size, SKU, and inventory for each
   - Set same price for all sizes (or price by size if preferred)

3. **Add Tags (CRITICAL):**
   - In the tags field, add: `drop-001`
   - This makes products appear on your shop page!

4. **Add Images:**
   - You'll need real product photography
   - Upload 3-5 images per product
   - First image is the thumbnail

5. **Publish:**
   - Set to "Active"
   - Save product

## Product Photography Checklist

For each product, you need:

### Required Shots:
- [ ] Hero shot (model wearing/using product)
- [ ] Back view (if apparel)
- [ ] Flat lay on neutral background
- [ ] Detail shot (logo, texture, tag)
- [ ] Lifestyle shot (in use on court/casual)

### Photography Style:
- Clean, minimal backgrounds (white, cream, concrete)
- Natural lighting (golden hour outdoor or soft studio)
- Brutalist aesthetic (geometric, stark, honest)
- Models in authentic poses (not overly styled)
- Show product in context (padel court, streetwear setting)
